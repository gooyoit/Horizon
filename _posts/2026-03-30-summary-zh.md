OpenAI 实施了一种客户端机器人检测机制，在允许用户输入前，验证 ChatGPT 的 React 应用是否已完全渲染和注水（hydrated）。该检查通过 Cloudflare 的机器人防护基础设施强制执行。 这种方法帮助 OpenAI 防止机器人将免费、未登录的 ChatGPT 访问滥用为非官方 API，从而将 GPU 资源留给真实用户。这也凸显了现代 AI 平台如何将网络安全深度集成到前端架构中。 该检测依赖于特定的 React 状态属性，这些属性仅在客户端完全注水后才存在；无头浏览器或未执行完整 JavaScript 包的机器人框架将无法通过此检查。这是一种应用层的机器人检测，而不仅仅是浏览器指纹识别。

hackernews · alberto-m · Mar 29, 20:21

**背景**: Cloudflare 提供多层机器人检测服务，涵盖网络层、浏览器层和应用逻辑层。OpenAI 为 ChatGPT 提供无需登录的免费访问，使其容易成为自动化爬虫和滥用的目标。为缓解这一问题，他们采用高级客户端完整性检查，验证用户环境的真实性。
