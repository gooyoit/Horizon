## [Mistral AI 发布用于验证代码生成的 Leanstral](https://mistral.ai/news/leanstral) ⭐️ 9.0/10

Mistral AI 推出了开源智能体系统 Leanstral，该系统利用 Lean 4 证明助手生成经过形式化验证的代码和数学证明。它能自主编写规范、实现代码并在单一框架内验证正确性。 Leanstral 将智能体 AI 与形式化方法相结合，使生成的软件不仅被编写出来，还能获得数学上的正确性保证——这对安全关键系统和可信 AI 至关重要。这种方法可能改变开发者构建可靠软件的方式，将验证直接嵌入编码工作流中。 Leanstral 使用 Lean 4 的类型检查器和证明引擎作为执行环境，使其能够根据可执行规范迭代优化代码。尽管目前专用于 Lean 4，但其架构展示了一种可推广到其他可形式化验证语言的模式。

hackernews · Poudlardo · Mar 16, 20:59

**背景**: Lean 4 是一种函数式编程语言，也是一个用于形式化验证的交互式定理证明器。在 Lean 中，程序和证明使用同一种语言编写，所有代码都必须通过严格的类型检查以确保逻辑一致性。形式化验证利用数理逻辑来证明软件行为完全符合规范，从而消除整类错误。智能体 AI 系统超越了文本生成，能够使用工具、执行代码并自主推理完成任务。

## [Mistral AI 发布统一版 Mistral Small 4 模型](https://simonwillison.net/2026/Mar/16/mistral-small-4/#atom-everything) ⭐️ 9.0/10

Mistral AI 发布了 Mistral Small 4，这是一个拥有 1190 亿参数的混合专家（MoE）模型，将推理（Magistral）、多模态（Pixtral）和智能体编程（Devstral）能力整合到一个采用 Apache 2.0 许可证的开源权重模型中。 这一发布意义重大，因为它提供了一个多功能、高性能的开源模型，将多种先进 AI 能力集成于一身，使开发者能用单一模型处理多样化任务，无需在多个专用模型间切换。Apache 2.0 许可证也进一步促进了其在商业和研究领域的广泛应用。 该模型采用混合专家架构，总参数量达 1190 亿，但每次推理仅激活 60 亿参数，并支持 'reasoning_effort' 参数，可在 'none'（轻量）或 'high'（深度推理）模式间切换以控制响应复杂度。模型已在 Hugging Face 上提供，体积为 242GB，并可通过 Mistral API 调用，但目前 API 文档尚未开放 reasoning_effort 参数设置。

rss · Simon Willison · Mar 16, 23:41

**背景**: 混合专家（Mixture-of-Experts, MoE）是一种神经网络架构，通过仅为每个输入激活模型参数的一个子集来提升效率，在保持高容量的同时降低计算成本。Mistral AI 此前曾分别发布过专注于推理的 Magistral、多模态的 Pixtral 和代码生成的 Devstral 等专用模型。此次将它们整合为单一模型，标志着向更通用人工智能系统的重要转变。

## [OpenAI Codex 新增子代理与自定义代理支持](https://simonwillison.net/2026/Mar/16/codex-subagents/#atom-everything) ⭐️ 9.0/10

OpenAI Codex 正式推出子代理（subagents）和用户自定义代理功能，用户可通过在 ~/.codex/agents/ 目录下的 TOML 文件进行配置。这些代理支持模型特定的任务委派和并行执行模式，与 Claude Code 中的实现类似。 该功能通过启用专业化、可组合的代理，显著提升了 AI 辅助软件开发中的工作流编排能力，从而提高效率和上下文管理效果。此举使 Codex 与 Claude、Gemini 和 VS Code 等平台所采用的智能体工程行业趋势保持一致。 自定义代理可指定指令和模型（包括专为快速推理优化的 gpt-5.3-codex-spark），并在提示中通过名称调用。默认子代理包括 'explorer'、'worker' 和 'default'，其中 'worker' 可能针对并行执行大量小型任务进行了优化。

rss · Simon Willison · Mar 16, 23:03

**背景**: 在 AI 工作流编排中，子代理是由主代理协调的专用组件，用于处理不同任务（如调试、代码追踪或 UI 修复），以提升模块化和性能。这种模式如今已广泛应用于 Claude Code、Gemini CLI 和 Visual Studio Code Copilot 等编码助手。OpenAI Codex 是 OpenAI 推出的 AI 驱动开发工具，可将自然语言转化为代码，并支持智能体工作流。
