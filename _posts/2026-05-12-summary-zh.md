## [Karpathy 预测 LLM 输出将从纯文本演进为交互式神经视频](https://twitter.com/karpathy/status/tweet-2053872850101285137) ⭐️ 8.0/10

Andrej Karpathy 分享了一种实用的提示技巧——要求 LLM 输出结构化 HTML 而非 markdown——并预测人机交互将从纯文本、经过 HTML，最终演进为由扩散模型直接生成的交互式神经视频和模拟。 Karpathy 的愿景重新定义了人们对 LLM 输出格式的思考方式，他指出视觉是大脑带宽最高的输入通道，AI 的输出应当越来越多地利用丰富的视觉和交互式媒体，而非纯文本。 Karpathy 还指出，当前的输入模态（音频、文本、视频）都不够用，未来的界面需要支持手势和指向等交互方式，就像人们面对面在屏幕前交流一样。他引用了一个近期走红的演示视频，作为交互式神经视频输出发展方向的一个参考。

twitter · Andrej Karpathy · May 11, 16:20

**背景**: 目前大多数 LLM 默认输出 markdown 格式，支持粗体、标题和表格等基本排版。要求 LLM 生成 HTML 是一种日益增长的做法，可以在浏览器渲染时解锁更丰富的布局、图形甚至交互性。像 StructEval 这样的研究基准已经开始系统性地评估 LLM 生成包括 HTML 和 SVG 在内的多种结构化输出的能力，反映出业界对此的日益关注。

## [Sam Altman 表示新 ChatGPT 模型已跨越质的门槛](https://twitter.com/sama/status/tweet-2053971387308745046) ⭐️ 8.0/10

OpenAI 首席执行官 Sam Altman 公开表示，新的 ChatGPT 模型、增强的个性化性格以及个性化功能的结合已经跨越了一个对他来说感觉全新的门槛。这表明 OpenAI 已经部署或即将发布一次重大的模型更新，在 AI 与用户交互的方式上有显著改进。 OpenAI 首席执行官的这一表态强烈暗示公司认为自己在用户体验方面实现了有意义的飞跃，而不仅仅是基准测试上的渐进式提升。这表明先进的个性化和更自然的 AI 性格可能成为 AI 行业的下一个主要竞争前沿，有可能改变数百万用户每天与 AI 助手交互的方式。 Altman 特别强调了三个协同作用的组成部分：新模型、性格增强和个性化能力。他使用的'对我来说跨越了门槛'这一措辞表明，这是来自首席执行官的主观但重要的定性判断，而非与特定指标或基准测试挂钩的声明。

twitter · Sam Altman · May 11, 22:51

**背景**: 自发布以来，OpenAI 一直在逐步增强 ChatGPT 的功能，包括记忆、自定义指令以及更自然的对话能力。AI 模型中的'性格'概念指的是模型交流时的自然程度和吸引力，而'个性化'则涉及 AI 记住用户偏好并相应地调整其回复。Altman 此前在预告重大产品里程碑时曾使用过类似的跨越门槛的说法，因此他的公开声明被视为即将发布的产品的风向标。

## [OpenAI 推出 Daybreak 计划，以 AI 驱动网络防御](https://twitter.com/sama/status/tweet-2053951874408276193) ⭐️ 8.0/10

Sam Altman 宣布推出 Daybreak，这是 OpenAI 的一项新计划，利用 Codex Security AI 智能体主动检测并修补软件漏洞，在攻击者利用之前加以防范。该平台旨在从软件开发之初就将网络安全融入其中，OpenAI 正积极寻求与尽可能多的公司展开合作。 Daybreak 标志着 OpenAI 向企业网络安全市场的重要产品扩展，直接与 Anthropic 的 Project Glasswing 展开竞争。随着 AI 在网络安全领域的能力快速提升，该计划有望通过大规模自动化漏洞检测与修复，从根本上改变企业持续保障软件安全的方式。 据报道，Daybreak 将 GPT-5.5 和 Codex Security 打包为一项托管网络安全计划，已拥有超过 20 个合作伙伴。该平台专注于在开发生命周期中将安全左移，实现持续防护而非被动修补。

twitter · Sam Altman · May 11, 21:34

**背景**: Codex Security 是 OpenAI 专门用于代码分析和安全任务的 AI 智能体，构建在其前沿语言模型之上。网络安全行业正越来越多地采用 AI 来执行漏洞扫描、威胁检测和自动修补等任务。Anthropic 的竞争项目 Project Glasswing 代表了 AI 实验室将模型应用于企业安全场景的类似尝试，表明前沿 AI 公司向网络安全领域扩展已成为更广泛的趋势。

## [OpenAI 宣布成立 OpenAI 部署公司](https://twitter.com/sama/status/tweet-2053904744331248112) ⭐️ 8.0/10

OpenAI 联合创始人 Greg Brockman 宣布成立「OpenAI 部署公司」，这是一个新的组织实体，旨在帮助企业成功实施 AI 部署并最大化其价值。Sam Altman 通过转发该公告表达了支持，显示出 OpenAI 最高领导层对此的强力认可。 此举标志着 OpenAI 商业模式的重大战略转变，从单纯提供 API 访问和模型转向提供企业级部署服务。这将使 OpenAI 能够在企业 AI 市场中获取更多价值，并与侧重咨询的 AI 服务提供商展开更直接的竞争。 该公告由 Greg Brockman（总裁/联合创始人）发布并由 Sam Altman（CEO）转发，表明这是一项最高优先级的战略举措。新实体的组织架构、定价模式和服务范围等完整细节在初始公告中尚未完全披露。

twitter · Sam Altman · May 11, 18:27

**背景**: OpenAI 历来专注于构建 GPT-4 等前沿 AI 模型，并通过 API 和 ChatGPT 等消费级产品提供服务。随着企业 AI 采用的加速，许多企业在有效部署 AI 方面面临实际挑战，包括集成、定制、安全和扩展等问题。创建专门的部署实体反映了 OpenAI 的认识：仅靠模型能力是不够的，客户还需要实施支持才能从 AI 中实现商业价值。
